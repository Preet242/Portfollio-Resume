Featured project

Athens Airbnb Market Intelligence Tool

Python / SQL / Streamlit / JMP  ·  preetp.me/airbnb  ·  Dec 2025

Airbnb pricing in Athens swings hard around UGA football. Most hosts either guess their rates or copy nearby listings, so they underprice gamedays and overprice the dead weeks. I built a tool that turns local demand signals into day-specific pricing guidance and competitive context.

What I built

  • Data collection: scraped and processed 460+ Airbnb listings via Apify, then cleaned and stored the dataset in SQLite using Python.
  • Feature engineering: built date-level features including weekends, seasonality, UGA football games, graduation, and academic calendar events.
  • Modeling: trained a multivariable linear regression to estimate dollar premiums and discounts tied to football weekends, graduation, and seasonal demand.
  • Validation: checked model significance and distributions in JMP.
  • Product layer: shipped an interactive Streamlit dashboard that outputs day-specific rate recommendations and competitive insight.
